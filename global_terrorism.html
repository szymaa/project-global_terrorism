Data source¶

Data comes from GTD (Global Terrorism Database) that contains information about more than 170,000 terrorist attacks worldwide from 1970 to 2016.

Data structure¶

The necessary information such as year, country, type of attack, target information etc. is stored in columns. To import data from csv file used pandas library and ISO-8859-1 encoding as well. Matplotlib and numpy librabries are also used to create charts.

data_terrorism=pd.read_csv('globalterrorismdb_0617dist.csv',encoding = 'ISO-8859-1', usecols=[1,7,8,9,10,28,29,34,35,36])

#Countries with the highest number of terrorist attacks (1970-2016)

attack_counts = data_terrorism['country_txt'].value_counts().nlargest(10)
plt.figure(facecolor='#F7F6F6', figsize=(15,6))
x = range(len(attack_counts))
y = attack_counts
plt.bar(x, y, color='#6690AD')
plt.title("Countries with the highest number of terrorist attacks (1970-2016)",fontsize=15)
plt.ylabel("Number of attacks")
plt.xticks(range(len(attack_counts)), attack_counts.keys())
plt.show()
